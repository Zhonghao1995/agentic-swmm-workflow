# Lessons Archived

Patterns whose ``confidence_score`` decayed below the dormant threshold
land here. Each block keeps its full ``aiswmm-metadata`` fence so a
maintainer can revive a pattern by moving the section back into
``lessons_learned.md`` (typically combined with fresh evidence — see
``aiswmm memory compact`` and issue #62).

This file is intentionally tracked by git so retirements stay
auditable: ``git log -- memory/modeling-memory/lessons_archived.md``
reconstructs the full provenance of every retired pattern.
